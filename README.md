# Scottish Auto Detailer Theme Preview

Premium, mobile-first concept pack for Scottish Auto Detailer.

## Structure

- 5 curated themes
- Each theme split into 4 focused subpages:
  - `overview`
  - `services`
  - `process`
  - `contact`
- 1 theme hub (`index.html`)
- Sticky bottom navigation for theme switching
- Local sub-navigation inside every theme page

- Theme 05: `theme-05-overview.html`, `theme-05-services.html`, `theme-05-process.html`, `theme-05-contact.html`

## Key files

- `index.html`: theme hub
- `site.css`: shared premium visual system and all theme variants
- `script.js`: bottom theme nav + active-state handling for local subnav
- `build-pages.mjs`: generator for all theme subpages

## Run locally

Open `index.html` in your browser.
